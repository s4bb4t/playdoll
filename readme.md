# AWS - Arina Web Services

## UPF -  User Profile Factory

Нужно реализовать **REST-API** сервис для **CRUD**'ов над юзерами. 

### `CRUD` - Create Read Update Delete

## Составляющие 
- Ручка (`POST` Эндпоинт) для **создания** юзера 
- Ручка (`GET` Эндпоинт) для **получения** юзера 
- Ручка (`PATCH` Эндпоинт) для **изменения** юзера 
- Ручка (`DELETE` Эндпоинт) для **удаления** юзера 



Профиль пользователя должен выглядеть так:
```go
type User struct {
	id int              // идентификатор пользователя
	name string         // имя пользователя
	email string        // почта 
	birthday time.time  // дата рождения 
	password string     // пароль
}
```

## Требования

- Возможность **Создать** пользователя надо только с **почтой** и **паролем**,
остальное можно указывать по желанию.


- Возможность **получить** пользователя по его идентификатору. Получать надо все, кроме пароля.


- Возможность **изменить** любое поле пользователя по его айди. Нужно сделать так, 
чтобы изменить можно было все указанные в запросе поля



- Возможность удалить пользователя по его айди. Но сделать удаление просто пометкой что юзер удален.


- Сервис должен быть персистентным - после перезапуска помнить все что было до перезапуска.
